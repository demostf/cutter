POV demo camera unlocker

Unlocks the camera in POV demos to allow the same controls as when viewing STV demos.

Select a demo file below to begin processing, once processed the file will be presented as download

Processing demo files can take a while

Note:

  • The demo still being recorded as POV demo means that only data around the recording player is in the demo file, players not near the recorder will behave weirdly
  • When the player respawns the freecam camera will be moved
  • The resulting demo might not work or crash tf2 on playback in rare cases
